# Narration Prompt Rules

톤:
- 밝고 친근함

속도:
- 약간 빠르게

감정:
- 초반은 놀람
- 중반은 설명
- 후반은 확신

발음:
- 또박또박

쉼:
- 문장 사이 짧게

브랜드 톤:
- 여성/남성 중 하나로 고정해서 브랜드 톤 유지

출력 형식:
1. Hook
2. 35초 나레이션 대본
3. TTS용 최종 문장
4. 감정/속도 가이드

필수 원칙:
- 반드시 사람이 읽는 것처럼 자연스럽게 작성
- TTS가 읽기 쉬운 문장
- 자막과 싱크가 맞도록 짧은 문장 사용
- 쉼표를 자연스럽게 사용
- 한 줄 10~15자 권장
